Insérer du css dans la page html 
        < link href="cheminDuFichierCss" rel="stylesheet">
Syntaxe de création 
        monSelecteur {{ '{' }} propriéter: value; {{ '}' }}

        h1 {{ '{' }}
          font-family: cursive;
          color: white;
        {{ '}' }}
Attribuer un ID a une balise 
        <div id="maDivBlue"> </div>

        #maDivBlue {{ '{' }}
          color: blue;
        {{ '}' }}
Attribuer une class 
        <div class="maDivBlue"> </div>

        .maDivBlue {{ '{' }}
          color: red;
        {{ '}' }}
Chaining, propriété de style en cascade 
        // séléctionne les balises p dans les div de class "maDivBlue"
        .maDivBlue p {{ '{' }}
        font-style: italic;
        {{ '}' }}

        // séléctionne les balises de class "maDivBlue" dans les balises p 
        p.maDivBlue {{ '{' }}
        text-align: center;
        {{ '}' }}
Réaction a la sourie 
        // le background des class "nav" changent de couleur quand la sourie passe dessus 
        .nav:hover {{ '{' }}
        background-color: green;
        {{ '}' }}
